# כחול עמוק, לב של אטישוק

Static PWA site.

Quick local test

```bash
# from the project root
python -m http.server 8000
# open http://localhost:8000
```

Publish to GitHub and enable Pages (automatic via Actions below):

```bash
git init
git add .
git commit -m "Initial PWA site"
git remote add origin https://github.com/<your-username>/<repo>.git
git branch -M main
git push -u origin main
```

The included GitHub Action will deploy the repository contents to the `gh-pages` branch when you push to `main`. Replace `<your-username>` and `<repo>` with your GitHub repo.
